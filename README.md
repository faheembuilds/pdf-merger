# PDF Merger

A desktop PDF merging tool built with Python and CustomTkinter.

## Features
- Select multiple PDF files
- Name your merged output file
- Clean dark UI
- Simple popup to save merged PDF

## Built With
- Python
- CustomTkinter
- pypdf

## How to Run
1. Install dependencies: `pip install customtkinter pypdf`
2. Run: `python Merger.py`
